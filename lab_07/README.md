# Лабораторна робота №7: Computer Vision та багатопотоковість у C++

## Опис завдання

Мета роботи - Навчитися поєднувати класичне програмування на C++ із сучасним CV/ML. Освоїти роботу з cv::dnn (Deep Neural Networks) для детекції об'єктів. Зрозуміти проблему блокування інтерфейсу важкими обчисленнями та вирішити її за допомогою std::thread та std::mutex, або std::condition_variable.

## Інструкція зі збірки та запуску

1. **Встановлення залежностей та завантаження моделі:**

   ```bash
   chmod +x preinstall.sh build.sh run.sh
   ./preinstall.sh
   ```

2. **Збірка проєкту:**

   ```bash
   ./build.sh
   ```

3. **Запуск програми:**

   ```bash
   ./run.sh
   ```

**Використання в програмі:**

- Натисніть **F** або **f**, щоб увімкнути/вимкнути режим детекції облич.
- Натисніть **ESC** або **q**, щоб вийти з програми.

## Короткий опис архітектури

Основна логіка розподілена між двома потоками:

- **Main Thread (UI)**: Відповідає за зчитування кадрів із камери (`cv::VideoCapture`), відображення кадру з рамкою (`cv::imshow`) та відслідковування натискань клавіш. Відправляє копію кадру детектору під м'ютексом і забирає координати знайдених облич.
- **Клас FaceDetector**: Реалізує Worker Thread для асинхронної детекції облич.
  - `std::thread worker`: Фоновий потік, що виконує інференс нейромережі (`net.forward()`).
  - `std::mutex mtx`: Захищає спільні дані (останній кадр та координати облич) від одночасного доступу з обох потоків.
  - `std::condition_variable cv`: Засіб для пробудження фонового потоку, коли новий кадр готовий до обробки.
  - `std::atomic<bool> running`: Прапорець для безпечного завершення роботи фонового потоку під час виходу з програми.
