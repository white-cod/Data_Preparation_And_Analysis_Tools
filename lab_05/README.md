# Лабораторна робота №5: Наука про дані: обмін результатами та початковий аналіз

## Опис завдання
Створення веб-додатку за допомогою бібліотеки Streamlit для візуалізації та аналізу даних вегетаційних індексів (VCI, TCI, VHI) з датасету NOAA (Лабораторна робота 2).

## Інструкція із запуску

1. Встановіть залежності:
    ```bash
    pip install -r requirements.txt
2. Завантажте дані (скрипт створить теку data і завантажить туди CSV файли):
    ```bash
    python download_data.py
3. Запустіть Streamlit додаток:
    ```bash
    streamlit run app.py
