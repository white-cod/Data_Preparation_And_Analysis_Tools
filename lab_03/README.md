# Лабораторна робота №3: Візуалізація даних

## Опис
Аналіз та візуалізація багатовимірного датасету Credit Approval (UCI Machine Learning Repository). 
Проведено очищення даних, побудовано графіки розподілу, кореляційні матриці та багатовимірні візуалізації за допомогою `matplotlib` та `seaborn`.

## Інструкція із запуску
1. Активуйте віртуальне середовище.
2. Встановіть залежності: `pip install -r requirements.txt`
3. Оновити компоненти: `!pip install --upgrade matplotlib seaborn pandas`
4. Запустіть Jupyter Notebook: `jupyter notebook`
5. Відкрийте `visualization.ipynb` та виконайте комірки.
