# Лабораторна робота №2: Наука про дані (підготовчий етап)

## Опис
Робота складається з двох частин:
1. Завантаження та обробка даних VHI-індексу з NOAA.
2. Аналіз датасету Individual Household Electric Power Consumption.

## Інструкція із запуску
1. Активуйте віртуальне середовище (створене в корені або локально)
2. Встановіть залежності: `pip install -r requirements.txt`
3. Запустіть Jupyter Notebook: `jupyter notebook`
4. Відкрийте `part1_noaa.ipynb` або `part2_power.ipynb` та виконайте комірки
