# Лабораторна робота №6: C++ OpenCV

## Опис завдання
Програма захоплює відео з веб-камери, дозволяє застосовувати фільтри в реальному часі за допомогою клавіатури, має повзунок для зміни яскравості та виводить лічильник FPS

## Вимоги до системи
- Компілятор: GCC/G++ з підтримкою C++17
- CMake: 3.10+
- Бібліотека: OpenCV (libopencv-dev)

## Інструкція із запуску
```bash
chmod +x *.sh
./preinstall.sh
./build.sh
./run.sh
```

## Керування
'n' - Нормальний режим (без фільтрів)

'i' - Інверсія кольорів

'b' - Gaussian Blur (Розмиття)

'c' - Canny (Виділення контурів)

ESC - Вихід з програми
