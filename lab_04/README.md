# Лабораторна робота №4: Візуалізація даних 2

## Опис завдання
Створення програми з інтерактивним інтерфейсом (matplotlib widgets) для малювання графіка гармоніки з накладеним шумом. Реалізовано можливість змінювати параметри гармоніки, шуму та фільтру.

## Інструкція із запуску (Windows 11)
1. Відкрийте термінал та перейдіть у теку `lab_04`.
2. Створіть та активуйте віртуальне середовище:
   ```cmd
   python -m venv venv
   .\venv\Scripts\activate
3. Встановіть залежності: pip install -r requirements.txt
4. Запустіть програму: python main.py

## Інструкція користувача
Слайдери Amplitude, Frequency, Phase: Змінюють параметри базової гармоніки.

Слайдери Noise Mean, Noise Covariance: Змінюють параметри генерації шуму. При цьому зміна параметрів гармоніки не генерує шум наново (він залишається "прив'язаним" до графіку).

Слайдер Filter Cutoff: Змінює частоту зрізу IIR-фільтра (Butterworth) для очищення зашумленого сигналу.

Чекбокси: Дозволяють вмикати/вимикати відображення шуму та відфільтрованого сигналу.

Кнопка Reset: Повертає всі параметри до початкових значень.
